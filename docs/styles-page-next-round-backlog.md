# 风格广场下一轮优化待办

## P1

- 增加收藏/稍后对比：让用户先标记 2-4 个候选风格，再进入对比视图。
- 增加风格对比表：对比适合项目、视觉关键词、平台端、颜色气质、落地风险。
- 增加详情快速预览抽屉：在列表页直接查看 CSS 变量、Prompt 和适用说明，减少跳转。

## P2

- 做一次完整无障碍检查：键盘焦点、读屏标签、颜色对比度、按钮可触达面积。
- 优化大列表性能：当风格数量继续增加时，引入分页、懒加载或虚拟列表。
- 增加排序能力：按最新、推荐、企业落地、安全保守、创意强度排序。

## P3

- 增加模板联动：从风格卡片直接跳到适配的模板、页面结构和组件清单。
- 增加团队说明字段：记录这个风格适合设计、产品、开发分别看什么。
